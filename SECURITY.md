# Security Policy

## Supported Versions

Only the latest versions are supported

| Version    | Supported          |
| ---------- | ------------------ |
| 1.12.x     | :white_check_mark: |
| < 1.12.0   | :x:                |

## Reporting a Vulnerability

Report vulnerabilities in the telegram channel https://t.me/instagrapi or https://github.com/adw0rd/instagrapi/issues
